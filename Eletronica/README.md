# Eletrônica

Esta pasta contém os arquivos relacionados ao sistema eletrônico dos robôs da equipe Challengers para a RoboCupJunior Soccer Lightweight 2026.

## Conteúdo

- Esquemas elétricos
- Diagramas de ligação
- Lista de componentes
- Datasheets dos sensores
- Informações sobre alimentação e reguladores de tensão
- Configuração dos motores e drivers

## Principais Componentes

- Arduino Mega 2560
- Sensor Infravermelho NIL 1046
- Bússola Magnética
- Sensores Ultrassônicos HC-SR04
- Sensores de Refletância QRE
- Bateria LiPo 11.1V
- Conversor DC-DC Buck Módulo Regulador Step Down 300W 20A ajustável
